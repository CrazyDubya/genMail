## Quick Start

### Prerequisites

- Node.js 20+
- pnpm 8+
- API keys for: Anthropic, OpenAI, Google AI, xAI, OpenRouter

### Setup

```bash
# Clone and install
git clone https://github.com/CrazyDubya/genMail.git
cd genMail
pnpm install

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Initialize database
pnpm run db:init

# Start development server
pnpm run dev
```
