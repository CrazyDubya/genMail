## Quick Start

### Prerequisites

- Node.js 20+
- npm (or pnpm/yarn)
- API keys for: Anthropic, OpenAI, Google AI, xAI, OpenRouter

### Setup

```bash
# Clone and install
git clone https://github.com/CrazyDubya/genMail.git
cd genMail
npm install

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Start development server
npx tsx src/server.ts
```

### First Run

1. **Test Model Connectivity**: The server will validate all configured models on startup
2. **Upload a Document**: POST to `/api/universe/create` with a PDF, TXT, or MD file
3. **Generate Emails**: The system processes documents â†’ extracts entities â†’ generates characters â†’ simulates world â†’ produces emails

### Environment Variables

```env
# Required
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_AI_API_KEY=...
XAI_API_KEY=...
OPENROUTER_API_KEY=...

# Optional
LOG_LEVEL=debug
MAX_CONCURRENT_TASKS=10
DEFAULT_EMAIL_COUNT=50
```
